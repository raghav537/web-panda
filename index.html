i made this because i wanted a chat thing that doesn’t phone home, doesn’t judge, and works with whatever model i’ve pulled. also pandas are chill. so: dodo panda — a wrapper around ollama that saves everything in your browser, talks to local llms, and generally minds its own business.

🧩

actually it's just

  • connects to localhost:11434 — your ollama
  • streams tokens like a gentle tap
  • auto saves chats into localStorage
  • lets you rename, pin, search (old chats don't vanish)
  • dark/light mode? yes, but default is just .. this
🥡

random things it does

  • regenerate a message (ctrl+enter)
  • export one chat as .md or full history as json
  • stop generation mid‑sentence (abort button)
  • see which model you're annoying right now
  • keyboard shortcuts that actually make sense

🍜models i've seen hanging around

llama3.2 codellama:latest mistral:v0.3 phi3:mini neural-chat deepseek-coder dolphin-phi qwen2.5 any:custom llama3.1:8b (plus whatever you `ollama pull`ed)

↳ it literally asks ollama what's available. if you just pulled something, hit refresh 🔄

🎚️ knobs you can tweak
temperature 0..2 top_k / top_p context 512–4096 num_predict repeat_penalty
⌨️ shortcuts (muscle memory stuff)
Enter send · Shift+Enter newline · Ctrl+N new chat
Ctrl+F settings · Ctrl+E export current chat
Ctrl+P pin/unpin · Esc stop generation
🧾 requirements? nothing fancy
  • ollama should be running ( ollama serve or app)
  • at least one model pulled (i like ollama pull llama3.2)
  • a browser that tolerates localStorage (chrome, firefox, safari, arc, etc)
  • that's it. no telemetry, no login, no api key.
✨ also it's fine if you just want to back up chats — there's a json export/import thing.
📦

backup? yeah one click

everything: conversations, pinned status, even model settings. download a .json, throw it on a usb or whatever. restore later. i do this before i clear browser data.
